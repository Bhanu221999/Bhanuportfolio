Face mask detection using Deep Learning

  • Developed a real-time face mask detection system to prevent the spread of COVID-19. Leveraged a basic Convolutional Neural Network (CNN) model using TensorFlow, Keras, Scikit-learn, and OpenCV.
  • The system processes webcam input, identifies faces, and classifies them as wearing or not wearing masks.
  • The proposed work contains three stages: pre-processing, Training a CNN, and Real-time classification. The pre-processing stage involves Grayscale Conversion of RGB image, image resizing and normalization, and the proposed CNN classifies faces with and without masks using Categorical cross-entropy as the loss function, achieving a Validation accuracy of 96%.
